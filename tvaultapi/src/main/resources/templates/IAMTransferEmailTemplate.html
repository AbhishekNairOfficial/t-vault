This message is to inform you that ownership of IAM Service Account has transferred from to you. You will now want to assign permissions to other users who you want to view or rotate the secret.
To do so please complete the following steps:

Create Access key:

  1. Login to T-Vault.
  2. Navigate to the "Service Accounts" tab.
  3. Select the "IAM Service Accounts" from dropdown.
  4. Click the "Create Access Key" link to create Access Keys for this IAM Service Account.

Assigning Permissions:

You can assign permissions to other users or groups you want to have permission to view or rotate the secret for the account.
  1. Login to T-Vault.
  2. Navigate to the "Service Accounts" tab and select "IAM Service Accounts" from dropdown.
  3. Select the IAM Service Account you want to manage.
  4. Select the "Permissions" button.
  5. Navigate to the Users tab and click on "Add User".
  6. Provide the user name you want to give permissions to and select either read, write or deny depending on the level of access you want to give.

Rotate Secret Key:

Once you are finished activating your IAM Service Account you can rotate the secret for the account.
  1. Login to T-Vault.
  2. Navigate to the "Service Accounts" tab and select "IAM Service Accounts" from dropdown.
  3. Select the IAM Service Account you want to manage.
  4. Select the secret folder.
  5. Select the "Rotate secret" link.
Please contact Cloud Support if you need assistance.
